Abstract base class providing xUnit contract tests for any type that inherits from Wolfgang.Etl.Abstractions.LoaderBase<TDestination, TProgress>.

Examples

public class MyLoaderContractTests
    : LoaderBaseContractTests<MyLoader, MyRecord, MyProgress>
{
    protected override MyLoader CreateSut(int itemCount) =>
        new MyLoader(connectionString);

    protected override IReadOnlyList<MyRecord> CreateSourceItems() =>
        new List<MyRecord> { new("a"), new("b"), new("c"), new("d"), new("e") };

    protected override MyLoader CreateSutWithTimer(IProgressTimer timer) =>
        new MyLoader(connectionString, timer);
}

Remarks

Inherit from this class to get a comprehensive suite of contract tests covering all public behaviour defined by Wolfgang.Etl.Abstractions.LoaderBase<TDestination, TProgress>:

  • All four LoadAsync overloads complete successfully.
  • Null guards on items and progress arguments.
  • Cancellation is honoured on all cancellable overloads.
  • CurrentItemCount is incremented as items are loaded.
  • ReportingInterval rejects values less than 1.
  • MaximumItemCount stops loading at the specified limit.
  • MaximumItemCount rejects values less than 0.
  • SkipItemCount skips the specified number of items.
  • SkipItemCount rejects values less than 0.
  • Progress callbacks fire when the timer fires.

You are responsible for implementing MaximumItemCount and SkipItemCount behaviour in your LoadWorkerAsync override.

CreateSourceItems()

Returns the source items used to feed the loader. Must return at least 5 items. The first itemCount items from this list are fed when CreateSut(itemCount) is called.
