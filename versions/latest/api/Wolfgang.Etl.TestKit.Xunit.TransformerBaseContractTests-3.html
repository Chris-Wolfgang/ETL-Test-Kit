Abstract base class providing xUnit contract tests for any type that inherits from Wolfgang.Etl.Abstractions.TransformerBase<TSource, TDestination, TProgress>.

public abstract class TransformerBaseContractTests<TSut, TItem, TProgress> where TSut : TransformerBase<TItem, TItem, TProgress> where TItem : notnull where TProgress : notnull

Examples

public class MyTransformerContractTests
    : TransformerBaseContractTests<MyTransformer, MyRecord, MyProgress>
{
    protected override MyTransformer CreateSut(int itemCount) =>
        new MyTransformer();

    protected override IReadOnlyList<MyRecord> CreateExpectedItems() =>
        new List<MyRecord> { new("a"), new("b"), new("c"), new("d"), new("e") };

    protected override MyTransformer CreateSutWithTimer(IProgressTimer timer) =>
        new MyTransformer(timer);
}

Remarks

Inherit from this class to get a comprehensive suite of contract tests covering all public behaviour defined by Wolfgang.Etl.Abstractions.TransformerBase<TSource, TDestination, TProgress>:

  • All four TransformAsync overloads yield the expected items.
  • Null guards on items and progress arguments.
  • Cancellation is honoured on all cancellable overloads.
  • CurrentItemCount is incremented as items are transformed.
  • ReportingInterval rejects values less than 1.
  • MaximumItemCount stops transformation at the specified limit.
  • MaximumItemCount rejects values less than 0.
  • SkipItemCount skips the specified number of items.
  • SkipItemCount rejects values less than 0.
  • Progress callbacks fire when the timer fires.

You are responsible for implementing MaximumItemCount and SkipItemCount behaviour in your TransformWorkerAsync override.
