Examples

var capture = new ProgressCapture<Report>();

await extractor.ExtractAsync(capture).ToListAsync();

ProgressAssert.HasReports(capture);
ProgressAssert.IsMonotonicallyIncreasing(capture, r => r.CurrentItemCount);
ProgressAssert.FinalReportSatisfies(capture, r => r.CurrentItemCount == 100);

Remarks

Like SynchronousProgress<T>, ProgressCapture<T> reports on the same thread that calls Report(T), making progress assertions in tests fully deterministic without delays or manual synchronisation. Unlike SynchronousProgress<T>, it also records every reported value so the full sequence can be inspected after the operation under test completes.

This type subsumes the manual new List<T>() plus new SynchronousProgress<T>(r => list.Add(r)) wiring that test code would otherwise repeat. Pair it with ProgressAssert for one-liner verification of the captured sequence.
