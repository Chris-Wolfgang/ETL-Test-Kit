Examples

var capture = new ProgressCapture<Report>();

await extractor.ExtractAsync(capture).ToListAsync();

ProgressAssert.HasReports(capture);
ProgressAssert.IsMonotonicallyIncreasing(capture, r => r.CurrentItemCount);
ProgressAssert.FinalReportSatisfies(capture, r => r.CurrentItemCount == 100);

Remarks

Each assertion throws an xUnit assertion exception (via the standard Assert.* API) when the condition is not met, so failures integrate with xUnit reporting exactly like any other assertion. These helpers replace the progress-validation boilerplate that downstream ETL test suites would otherwise copy between projects.
