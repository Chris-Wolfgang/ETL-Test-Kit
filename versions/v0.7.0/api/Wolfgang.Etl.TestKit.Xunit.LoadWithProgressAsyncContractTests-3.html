Abstract base class providing xUnit contract tests for any type that implements Wolfgang.Etl.Abstractions.ILoadWithProgressAsync<TDestination, TProgress>.

public abstract class LoadWithProgressAsyncContractTests<TSut, TItem, TProgress> where TSut : ILoadWithProgressAsync<TItem, TProgress> where TItem : notnull where TProgress : notnull

Type Parameters

TSut

The type under test. Must implement Wolfgang.Etl.Abstractions.ILoadWithProgressAsync<TDestination, TProgress>.

TItem

The type of item the loader consumes.

TProgress

The type of the progress report.
